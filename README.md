# 💼 Job Scraper

A Python-based job scraping application that fetches job listings and analyzes required skills using API integration.

## 🚀 Features
- Fetch job listings from API
- Extract and analyze required skills
- Predefined skill matching system
- Simple backend using Flask

## 🛠️ Tech Stack
- Python
- Flask
- REST API
- JSON

## 📂 Project Structure
- app.py → Main application
- scraper.py → Data scraping logic
- skills.py → Skill database

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add your API key in app.py
4. Run:
   python app.py

## 🔐 Security Note
API keys are not included in this repository.
